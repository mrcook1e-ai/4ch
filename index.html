Zapret —
без VPN, навсегда

Открытый инструмент обхода DPI-блокировок. Работает на уровне сети — не тратит трафик, не меняет IP, не тормозит соединение. Discord, YouTube, Telegram.

Антивирус может сработать на WinDivert.sys — это ложное срабатывание. Добавь папку в исключения или отключи «Обнаружение PUA».
Скачать последний релиз Исходный код на GitHub
Авторы

Инструмент: @Flowseal
Ядро zapret: @bol-van

01
Включи Secure DNS
Chrome → Настройки → Конфиденциальность → «Безопасный DNS». Выбери любого кроме дефолтного.
dns.google/dns-query
02
Скачай архив
Со страницы релизов. ПКМ по архиву → Свойства → галочка «Разблокировать».
03
Распакуй без кириллицы
Путь не должен содержать кириллицу, пробелы или спецсимволы.
C:\zapret\
04
Запусти стратегию
Пробуй general.bat по одному. Рабочий — ставь на автозапуск.
service.bat → Install
05
Если войс не работает
Обнови адреса голосовых серверов Discord через service.bat.
service.bat → Hosts
